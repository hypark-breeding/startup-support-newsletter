# Source Policy

## Priority

Use official sources first.

1. Program owner or application portal.
2. National portals, central ministries, and public institutions.
3. City/province, municipal, or affiliated public organizations.
4. National public aggregators such as Bizinfo, K-Startup, SMEs24, KONEPS, or Data.go.kr.
5. Private aggregators only as leads.

When the user does not provide a URL, discover candidate official URLs first and clearly label any unverified lead sources.

## Verification

Every user-facing recommendation should include:

- official source URL when available
- application period
- organization
- eligibility summary
- verification date
- confidence level

## Confidence

- `high`: official page confirms title, deadline, eligibility, and application route.
- `medium`: official source confirms most fields, but one field requires interpretation.
- `low`: source is an aggregator or partial official page; requires manual confirmation.

## Exclusions

Exclude:

- closed programs unless the user asks for historical context
- pure press releases with no application path
- hiring notices unless the user asks for startup hiring support
- login-only content unless approved by the user

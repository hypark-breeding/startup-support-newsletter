# Newsletter Format

## Title

`서울 창업지원사업 브리핑 - YYYY-MM-DD`

## Sections

1. This Week's Open Programs
2. Closing Soon
3. New or Notable Sources
4. District Spotlight
5. Watchlist

## Item Format

```markdown
### Program Title

- Organization:
- Application period:
- Region:
- Target:
- Benefit:
- Why it matters:
- Official link:
- Verified at:
- Confidence:
```

## Tone

Use concise Korean for founders. Avoid bureaucratic phrasing when summarizing, but keep
official program names and eligibility conditions precise.
